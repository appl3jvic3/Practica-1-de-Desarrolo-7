# Practica-1-de-Desarrolo-7


/práctica en **PHP** con ejemplos básicos de programación.  
Incluye archivos sencillos para trabajar conceptos como variables, operadores matemáticos, precedencia de operadores y estructuras de repetición.

## Descripción

Este proyecto contiene ejercicios introductorios en PHP orientados a:

- Uso de variables
- Comprobación de tipos
- Operaciones matemáticas
- Precedencia de operadores
- Bucles `for` y `foreach`

Cada archivo presenta un ejemplo independiente en formato HTML + PHP para facilitar su ejecución en un servidor local.

## Tecnologías usadas

- **PHP**
- **HTML**

## Estructura del repositorio

- `Practica_Variables.php`  
  Ejemplo de uso de variables y validación de tipos con `is_integer()` e `is_string()`.

- `OperacionesMat.php`  
  Ejemplos de operaciones matemáticas básicas:
  suma, resta, multiplicación, división, módulo y exponente.

- `Orden_de_Precedencia.php`  
  Muestra cómo funciona la precedencia de operadores en expresiones matemáticas.

- `ForEach.php`  
  Ejemplos de uso de `for` y `foreach`.

## Requisitos

Para ejecutar estos archivos necesitas:

- Un servidor con soporte para **PHP**
- Un navegador web

Puedes usar, por ejemplo:

- XAMPP
- WAMP
- MAMP
- Laragon
- PHP embebido

## Cómo ejecutar el proyecto

### Opción 1: Servidor local
1. Clona o descarga el repositorio.
2. Copia los archivos en la carpeta pública de tu servidor:
   - `htdocs` en XAMPP
   - `www` en WAMP
   - `public_html` en otros entornos
3. Abre el navegador y accede al archivo deseado.
